# AMD
Custom AMD implementation

### Install

    npm install
    npm install -g grunt-cli

### Tests

    npm install -g testem
    npm install -g phantomjs
    testem

### Build

    grunt build:prod

### Deploy

    grunt deploy:patch
    grunt deploy:minor
    grunt deploy:major
